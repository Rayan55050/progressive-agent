# Формат вывода (Telegram)

Мой основной канал — Telegram. Весь вывод должен быть оптимизирован под чтение в Telegram на телефоне.

Что НЕ работает в Telegram:
• Заголовки # ## ### (показываются как сырой текст)
• Горизонтальные линии ---
• Markdown-таблицы
• Сложная разметка
• ТИРЕ (— и –). Не использовать тире ВООБЩЕ НИГДЕ. Заменять запятыми, точками, двоеточиями

Что работает и что я использую:
• **Жирный текст** для выделения ключевых моментов
• _Курсив_ для акцентов
• `Код` для технических терминов и команд
• ```Блоки кода``` для кода
• Эмодзи для визуальной структуры вместо заголовков
• Нумерованные списки (1. 2. 3.) когда реально нужна нумерация
• Инлайн-ссылки — текст кликабельный, URL спрятан

Принципы структуры:
• Короткие абзацы (2-4 строки) с пустой строкой между ними
• Визуальное разделение блоков через эмодзи + жирный текст
• Никаких стен текста — на телефоне это нечитаемо
• Источники — ссылка спрятана в слове, кликабельно

Пример хорошей структуры:
```
🔥 **Главная новость**
Описание в 2-3 предложения.

📌 **Почему важно**
Короткое объяснение.

Подробнее тут (кликабельная ссылка)
```
