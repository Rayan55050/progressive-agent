# Skill Creator — самообучение

Ты умеешь создавать новые скиллы для себя. Это как расширение собственных возможностей.

## Как создать новый скилл

1. Создай папку и файл SKILL.md через `file_write`
2. Путь: `skills/{skill_name}/SKILL.md` (в корне проекта)
3. Формат — YAML frontmatter + Markdown инструкции

## Шаблон SKILL.md

```yaml
---
name: имя_скилла
description: "Краткое описание"
tools:
  - список_тулов_которые_нужны
trigger_keywords:
  - ключевое_слово_1
  - ключевое_слово_2
---

# Название скилла

Инструкции для себя: когда и как использовать этот скилл.

## Инструменты
Описание каждого тула и его параметров.

## Правила
Ограничения и правила использования.
```

## Доступные тулы для новых скиллов

Тулы, которые уже зарегистрированы и которые можно использовать:

### Поиск в интернете
- `web_search` — быстрый поиск (query, topic, max_results)
- `web_extract` — извлечение контента из URL
- `web_research` — глубокий ресёрч

### Файлы
- `file_search`, `file_read`, `file_list`, `file_write`, `file_delete`
- `file_open`, `file_send`, `file_pdf`, `file_copy`

### Email
- `email_inbox`, `email_read`, `email_compose`

### CLI
- `cli_exec` — выполнение команд (command, shell, working_dir)

### STT
- `stt` — распознавание голоса

## Правила создания скиллов

1. **Имя папки** = имя скилла (латиница, snake_case)
2. **trigger_keywords** — минимум 3-5 ключевых слов на русском И английском
3. **tools** — только из списка доступных тулов выше. Нельзя ссылаться на несуществующие
4. **Инструкции** — конкретные, с примерами. Ты пишешь инструкции ДЛЯ СЕБЯ
5. **Правила** — обязательно добавь секцию с ограничениями
6. **После создания** — скажи владельцу, что скилл создан и будет доступен после перезапуска

## Пример: создание скилла для крипты

Если владелец скажет "научись отслеживать биткоин":

1. Создай `skills/crypto/SKILL.md` через file_write с:
   - tools: [web_search]
   - keywords: [биткоин, btc, крипта, курс, bitcoin]
   - Инструкции: как искать курсы, где брать данные, формат ответа

2. Скажи: "Готово! Создал скилл crypto. После рестарта смогу отслеживать крипту."

## ВАЖНО

- Ты НЕ можешь создавать новые тулы (Python-код). Только SKILL.md файлы
- Новые скиллы работают через КОМБИНАЦИЮ существующих тулов
- Это как набор инструкций, которые говорят тебе КАК использовать тулы для конкретной задачи
- Скилл активируется после перезапуска бота (владелец должен перезапустить)
