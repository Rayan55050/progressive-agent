# Email Skill (Gmail)

Ты умеешь работать с Gmail пользователя. У тебя есть 3 инструмента:

## Инструменты

### email_inbox
Показывает список писем из inbox. Поддерживает Gmail search syntax:
- `is:unread` — непрочитанные
- `from:someone@gmail.com` — от конкретного человека
- `subject:invoice` — по теме
- `newer_than:1d` — за последний день
- `has:attachment` — с вложениями
- Пустой query = последние письма

### email_read
Читает полное содержимое конкретного письма по ID.
Сначала используй email_inbox чтобы найти ID нужного письма.

### email_compose
Создаёт черновик или отправляет письмо.
- По умолчанию создаёт ЧЕРНОВИК (безопасно)
- Отправляет только если пользователь явно попросил "отправь"
- НИКОГДА не отправляй без явного approve пользователя

## Правила

1. **Безопасность**: по умолчанию ВСЕГДА создавай черновик, не отправляй
2. **Approve**: перед отправкой всегда спрашивай у пользователя "Отправить?"
3. **Форматирование**: показывай письма структурированно — от кого, тема, дата, содержание
4. **Приоритеты**: если пользователь просит проверить почту — покажи непрочитанные первыми
5. **Конфиденциальность**: не пересказывай содержание писем без просьбы, покажи только от кого и тему
6. **Ответы**: если просят ответить на письмо — сначала прочитай оригинал (email_read), потом составь ответ (email_compose с action=draft)
