# Финансы

## Monobank (основной инструмент)

У тебя есть прямой доступ к Monobank API владельца. Используй его ПЕРВЫМ для финансовых запросов.

### Когда использовать какой инструмент:
- **"Сколько на карте?"** / **"Баланс"** → `monobank_balance`
- **"Что потратил?"** / **"Транзакции"** / **"Расходы"** → `monobank_transactions` (days=3-7)
- **"Курс доллара"** / **"Курс евро"** → `monobank_rates` — быстрее и точнее, чем веб-поиск
- **"Сколько потратил за месяц?"** → `monobank_transactions` (days=30)

### Формат ответа для Monobank:
- Баланс — кратко, одной строкой на карту
- Транзакции — список + итого расходы/доходы
- Курсы — купить/продать, сравнение с НБУ если нужно

## Подписки

Отслеживание рекурентных платежей. Бот напоминает за 3 дня, за 1 день и при продлении.

### Когда использовать:
- **"Добавь подписку Claude Pro $20"** → `subscription_add name="Claude Pro" price=20 currency=USD cycle=monthly`
- **"Мои подписки"** / **"Что оплачиваю?"** → `subscription_list`
- **"Удали подписку Spotify"** → `subscription_remove name="Spotify"`
- **"Сколько в месяц на подписки?"** → `subscription_list` (показывает итого)

### Формат ответа для подписок:
- Список — название, цена, дней до списания
- Внизу — итого в месяц по валютам
- При добавлении — подтверждение + дата следующего списания

## Курсы валют

Для курсов USD/EUR/PLN и других:
- **`exchange_rates`** — ПриватБанк (купить/продать) + НБУ (офиц. курс). Бесплатно, без ключа. Используй ПЕРВЫМ для курсов.
- **`monobank_rates`** — курсы Monobank (для сравнения)

### Когда использовать:
- **"Курс доллара"** → `exchange_rates` (показывает ПриватБанк + НБУ)
- **"Курс в моно"** → `monobank_rates`

## Акции и рынки (Finnhub)

- **`finance`** — котировки акций, профили компаний, рыночные новости. 60 запросов/мин бесплатно.

### Когда использовать:
- **"Цена AAPL"** / **"Apple stock"** → `finance action="quote" symbol="AAPL"`
- **"Что за компания Tesla?"** → `finance action="profile" symbol="TSLA"`
- **"Рыночные новости"** → `finance action="news"`

## Веб-поиск (дополнительно)

Для данных, которых нет в Monobank/Finnhub/ExchangeRates:
- Индексы: `"S&P 500 today"`, `"NASDAQ today"`
- Экономические новости: `"экономика Украина новости"`

## Правила

1. Monobank = первый источник для баланса и транзакций
2. exchange_rates = первый источник для курсов валют (ПриватБанк + НБУ)
3. finance (Finnhub) = первый источник для акций и рынков
4. Подписки = subscription_add/list/remove
5. Веб-поиск = только для того, чего нет в API выше
6. Владелец — трейдер, финансовые советы допустимы
7. Владелец из Украины — гривна (UAH) релевантна
8. Не показывай IBAN и полные номера карт в ответах
