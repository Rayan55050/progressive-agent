# Getting Help

If you need help with Progressive Agent, here are the best ways to get support.

## Bug Reports

Found a bug? [Open a GitHub Issue](https://github.com/progressiveai/progressive-agent/issues/new?template=bug_report.md).

Please include:
- Steps to reproduce the problem.
- Expected behavior vs. what actually happened.
- Python version (`python --version`) and operating system.
- Relevant log output (redact any API keys or personal data).

## Questions & Discussions

Have a question about setup, architecture, or how to build a new tool/skill?

- **[GitHub Discussions](https://github.com/progressiveai/progressive-agent/discussions)** -- the best place for questions, ideas, and general conversation.
- Check the [FAQ](FAQ.md) first -- your question may already be answered.
- Check [TROUBLESHOOTING.md](TROUBLESHOOTING.md) for common issues and solutions.

## Feature Requests

Want to suggest a new tool, skill, or improvement?

- Open a [GitHub Discussion](https://github.com/progressiveai/progressive-agent/discussions/categories/ideas) in the Ideas category.
- Or open a [GitHub Issue](https://github.com/progressiveai/progressive-agent/issues/new?template=feature_request.md) with the "enhancement" label.

## Contributing

Want to contribute code, documentation, or skills?

- Read the [Contributing Guide](CONTRIBUTING.md) for setup instructions and code style guidelines.
- Read the [Design Decisions](DESIGN_DECISIONS.md) to understand the project's architectural choices.
- Check [open issues](https://github.com/progressiveai/progressive-agent/issues) for tasks labeled `good first issue` or `help wanted`.

## Security Issues

If you discover a security vulnerability, please **do not open a public issue**. Instead, email [contact@progressiveai.me](mailto:contact@progressiveai.me) with details. We will respond promptly and coordinate a fix before public disclosure.
